Weather App — React.js & OpenWeatherMap API

Developed a responsive weather application that allows users to fetch real-time weather data for any city worldwide. The app integrates the OpenWeatherMap API to retrieve live weather conditions, including temperature, humidity, wind speed, and weather descriptions. Implemented clean UI/UX using React state management and Axios for asynchronous API requests. Key Features: Search weather by city name Real-time temperature and weather condition display Error handling for invalid or unknown cities Responsive design for all screen sizes Units displayed in Celsius for global user understanding GitHub Repo: Weather App
